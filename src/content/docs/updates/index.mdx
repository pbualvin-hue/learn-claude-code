## 🔄 這一區在做什麼

Claude Code 更新得很勤（常常一週好幾版）。官方的 [CHANGELOG](https://raw.githubusercontent.com/anthropics/claude-code/main/CHANGELOG.md) 是全英文流水帳，混著大量內部修正。

這一區幫你做三件事：

1. **翻譯重點**：每批更新用白話講「對日常使用有沒有感」。
2. **過濾雜訊**：內部修正歸到「可略過」，你只需要看有感的部分。
3. **同步本站**：列出這批更新影響了本站哪些教學頁，讓內容跟著版本走。

頁面清單會依版本由新到舊排在左側側欄。

## ⚠️ 限制與注意

- 本站更新頻率上限為**每週一次**（排程）＋不定期手動觸發，不做即時跟版——看到官方出了新版而這裡還沒有，是正常的，不是壞掉。
- 每篇解讀的依據是官方 changelog 與文件；changelog 只有一句話、官方文件查不到細節的行為，會標「⚠ 未驗證」。
- 想知道目前全站內容以哪一版為基準，看每頁最下方 footer 的「內容基準版本」。
