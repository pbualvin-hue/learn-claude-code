學開車不會先把交通法規整本背完才上路。

你會直接坐進駕駛座，遇到紅綠燈才學紅綠燈，需要倒車才學倒車。

這個軌道用同樣的邏輯設計：你會做一個真的看得到成果的小專案，功能在你**實際卡關的當下**才登場，不是先背完一輪術語再開始動手。

## 🎮 這個軌道怎麼玩

- **做中學**：全程圍繞一個小專案進行，不是先聽一輪課才動手。
- **卡關才學功能**：專案途中會自然撞到幾個「卡住的時刻」——它跳出一個許可確認、它每次都忘記你講過的偏好。每次卡關，才順便學會對應的那個功能。
- **每一站沉澱一條規則**：解決卡關之後，你會留下一條屬於自己的小規則（通常是寫進 `CLAUDE.md`，或至少記在腦子裡）。這條規則，才是這個軌道真正想留給你的東西，功能清單本身不是重點。

## 🤝 這個軌道保證什麼、不保證什麼

我們保證帶你走完這個範本專案。

我們**不**保證你之後自己動手的專案一定會順利完成——每個人的專案不一樣，遇到的狀況也不一樣。

但走完這裡之後，你會有一套「卡關 → 怎麼問 → 發現功能 → 沉澱規則」的方法可以帶著走，這才是真正可攜帶的東西（走完軌道後可以看[下一步地圖](next-step-map/)，裡面把這件事講得更完整）。
