## 🎓 這是什麼

Anthropic Academy（`anthropic.skilljar.com`）是 Anthropic 官方開的線上課程平台。

用的是第三方教學系統 Skilljar，不是 `docs.claude.com` 那個文件站，也跟本站沒有任何隸屬關係。

本站只是幫你導覽，內容本身完全由 Anthropic 製作與擁有。

截至查證當下（2026-07-13），平台上約有 20 門課，分成幾大類：

- **Claude Code 相關**：`Claude Code 101`、`Claude Code in Action`、`Introduction to Agent Skills`、`Introduction to Subagents`、`Introduction to Model Context Protocol`（及進階版）
- **一般 Claude 使用與開發**：`Claude 101`、`Claude Platform 101`、`Building with the Claude API`
- **雲端整合**：`Claude with Amazon Bedrock`、`Claude with Google Cloud's Vertex AI`
- **AI Fluency 系列**（給教育者、學生、企業、非營利組織等不同受眾的 AI 素養課程，非 Claude Code 專屬）
- 其他如 `Introduction to Claude Cowork`（另一個產品線的操作課，跟 Claude Code 不是同一個工具）

適合誰上：想要有結構、有進度、有完課證明的人。

如果你偏好邊做邊查，本站的 [學習路徑](../../learn/getting-started/) 加 [功能參考](../../reference/what-is-claude-code/) 已經涵蓋大部分日常情境，不一定需要另外上課。

**怎麼進去上課**：直接到官網 [anthropic.skilljar.com](https://anthropic.skilljar.com) 註冊帳號（Skilljar 帳號，跟你的 Claude 帳號是分開的兩件事），挑課、報名、開始上課即可，不需要透過本站中轉。

:::caution[本站不轉載課程內容]
Academy 的課程內容（影片、講義、練習題）沒有公開授權聲明，等同保留所有權利。

本頁只寫「這些課大概教什麼、跟本站哪裡對得上」，不會翻譯、改寫或摘錄任何一堂課的實際教材。

想學內容，請直接去官網上課。
:::

## 📜 證照 vs 只學內容

本站的定位是幫你把內容學會、能實際上手用 Claude Code，不代辦證照、也不追求把 Academy 每一張證書集滿。

如果你的目的是履歷上要有張證書，直接去官網上課考證。

如果你只是想學會怎麼用，本站的學習路徑加 Academy 挑幾門對得上的課搭配著看，效率通常更好。
