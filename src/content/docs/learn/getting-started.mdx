## 🎯 這一課只做一件事

把 Claude Code 裝好，然後跟它講到第一句話。就這樣，不用多。

慢慢來。看不懂的地方先跳過沒關係——真正重要的東西，後面用到的時候會再講一次。

## 🖥️ 它長什麼樣子？

先講一句可能讓你鬆一口氣的話：Claude Code **沒有**漂亮的視窗、也沒有一排按鈕。

它住在一個叫「終端機」的地方——一個只能打字的小視窗（黑底或白底都有可能）。第一次看到會有點陌生，這很正常，大家都一樣。

你可以這樣想：終端機就像一個聊天視窗，只是對面坐的是一個很懂電腦的幫手。你打字告訴它要做什麼，它看懂了就動手，而且動手前還會先問過你。

啟動之後，畫面大概長這樣：

<TerminalStartup />

終端機本身不是 Claude Code，是你電腦本來就內建的東西（Windows 叫它 PowerShell，Mac 叫它 Terminal）。Claude Code 只是住進去的一個幫手，打 `claude` 就能把它叫出來。

## 📦 先把它裝起來

安裝要打的那一行指令，我們放在另一頁隨時更新：[安裝與登入](../../reference/install/)。

為什麼不直接寫在這裡？因為官方偶爾會調整，寫死在這反而可能害你照到過期的版本。那頁會一直跟著最新的走。

大概就三步：

<Steps>

1. 打開終端機（Windows 在開始選單搜「PowerShell」，Mac 用 Cmd＋空白鍵搜「Terminal」）。

2. 到 [安裝與登入](../../reference/install/)，複製你系統對應的那一行，貼上、按 Enter。

3. 裝好後打 `claude --version`，看到印出一串版本號，就成功了。

</Steps>

:::note[電腦太舊會不會裝不了？]
門檻其實不高，這幾年買的電腦幾乎都可以。詳細規格列在 [安裝與登入](../../reference/install/)，真的不確定再去看就好，先別卡在這裡。
:::

## 💬 第一次叫出 Claude

裝好了，找一個你想讓它幫忙的資料夾（桌面上隨便一個資料夾也行），在那裡打開終端機，輸入：

```bash
claude
```

第一次會有兩個小關卡，別緊張，都很快：

- **登入**：會跳出瀏覽器，要你登入 Claude 帳號。
  小提醒：Claude Code 需要付費方案（Pro／Max／Team 之類）才能用，免費的 claude.ai 帳號不含這個功能。帳號請自己到官網辦，密碼不要交給任何人、任何程式。
- **問你信不信任這個資料夾**：第一次在某個資料夾裡用，它會先問過你一次。這是它故意設計的安全關卡，不是出錯，點信任就好。

兩關過了，你就會看到上面那張示意圖的歡迎畫面，停在那裡等你打字。

:::tip[不用每次都登入]
登入一次就記住了，下次打 `claude` 直接就能用。哪天想換帳號，再打 `/login` 就好。
:::

## ✍️ 跟著做

現在換你動手：

<Steps>

1. 照 [安裝與登入](../../reference/install/) 裝好，打 `claude --version` 確認有版本號。

2. 找一個空資料夾，在那裡打開終端機，輸入 `claude`。

3. 跳出登入就登入；問你信不信任資料夾，選信任。

4. 看到畫面後，直接打字問它一句：

   ```text
   這個資料夾裡有什麼？
   ```

   **你會看到**：它讀了資料夾，用中文告訴你裡面有什麼。如果資料夾是空的，它會說「沒有東西」——這也對，代表它真的在看，不是隨便回你。

</Steps>

<details>
<summary>卡住了？點開看</summary>

- **打 `claude` 說找不到指令**：多半是還沒裝好，或終端機需要重開。關掉視窗、重開一個再試；還是不行就回 [安裝與登入](../../reference/install/) 對一下步驟有沒有漏。
- **卡在登入畫面，回終端機還是沒登入**：把瀏覽器那頁關掉，重打一次 `claude`，通常會重新跳登入。
- **畫面停住不動**：多半不是壞掉，是它在等你打字。看一下最後一行是不是在問你 yes/no。
- 都試過還是卡，就看 [常見問題](../../reference/faq/)。

</details>
