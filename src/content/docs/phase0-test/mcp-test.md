本頁為 Phase 0 搜尋驗證測試頁，正式上線前會移除。

MCP（Model Context Protocol）是讓 Claude Code 連接外部服務的協定。
透過 mcp 連接器，你可以讓 Claude 讀取 Google Drive、操作瀏覽器、查詢資料庫。

排程功能可以搭配 MCP 使用：例如每週日晚上自動讀取更新日誌。

中文斷詞測試句：初學者最常問的問題是資料安全，
你的檔案內容只有在對話中使用時才會被讀取。
