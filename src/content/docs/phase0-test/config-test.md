本頁為 Phase 0 搜尋驗證測試頁，正式上線前會移除。

`/config` 是 Claude Code 的設定指令，輸入後會開啟互動式設定面板，
可以調整主題、模型、通知等選項。

settings 檔案則存放更進階的設定：權限清單、環境變數、hooks 定義。
記憶功能（CLAUDE.md）讓 Claude 記住你的偏好，每次對話都自動載入。

中文斷詞測試句：如果它一直問你要不要允許，
可以到權限設定裡把常用的指令加入允許清單。
