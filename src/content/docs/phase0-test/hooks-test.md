本頁為 Phase 0 搜尋驗證測試頁，正式上線前會移除。

hooks 是 Claude Code 的自動化掛鉤機制。當你想在特定事件發生時自動執行指令，
可以在 settings 裡設定 hooks。常見用途包括：格式化程式碼、發送通知、記錄操作。

權限設定與 hooks 息息相關：hooks 執行的指令不會再次詢問權限，
所以設定前要想清楚安全性。

中文斷詞測試句：這個網站教你怎麼使用排程功能自動更新內容，
並且用記憶功能保存你的偏好設定。
